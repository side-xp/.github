# .github

This repository contains organization-wide defaults and documentation for all repositories under [@side-xp](https://github.com/side-xp).

## Contents

- **Community health files**: default [`CONTRIBUTING.md`](./CONTRIBUTING.md), [`SUPPORT.md`](./SUPPORT.md), [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) inherited by all repositories
- **Issue & PR templates**: default templates for bugs, features, documentation
- **Reusable workflows**: shared CI/CD pipelines under `workflows/`
- **Coding standards**: organization-wide conventions under [`docs/coding-style/`](./docs/coding-style/README.md)

## How defaults work

Files in this repository act as fallbacks for any repository in the organization that does not define its own version. See [GitHub's documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for the full list of supported default files.

## Contributing

To propose changes to organization-wide standards, open an issue or PR here. Significant changes (coding standards, workflows) should be discussed before merging as they affect all repositories.
